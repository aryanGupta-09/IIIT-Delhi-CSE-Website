## Installation

1. Clone the repo
```bash
  git clone https://github.com/aryanGupta-09/IIIT-Delhi-CSE-Website.git
```

2. Go to the project directory
```bash
  cd IIIT-Delhi-CSE-Website
```

3. Install NPM packages
```bash
  npm install
```

4. Start the development server
```bash
  npm run dev
```
